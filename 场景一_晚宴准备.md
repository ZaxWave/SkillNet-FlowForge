       # 场景一：晚宴准备（多目标并行）

**总任务：** "加热两片面包放餐桌，冷却一瓶酒放边桌，清洗盘子和叉子放餐桌，扔掉变质食物。"

---

## 第一阶段：理解 + 扫描（串行）

```
alfworld-goal-interpreter  →  拆解为 4 个子目标
alfworld-environment-scanner → 构建房间物品地图
```

拆解结果：

- 子目标1：加热面包×2 → 餐桌
- 子目标2：冷却红酒×1 → 边桌
- 子目标3：清洗盘子×1 + 叉子×1 → 餐桌
- 子目标4：丢弃变质食物 → 垃圾桶

---

## 第二阶段：四路并行

```
          ┌─────────────┬─────────────┬─────────────┐
          ▼             ▼             ▼             ▼
      【分支A】      【分支B】      【分支C】      【分支D】
      面包×2加热    红酒冷却      餐具清洗       清理变质食物
```

### 分支A — 面包加热（×2）

```
object-locator → receptacle-opener → inventory-management(循环×2)
  └─ 循环体: object-retriever → appliance-navigator(microwave)
             → object-heater → location-navigator(diningtable) → object-placer
→ task-verifier
```

**用到的 skill（10个）：** object-locator, receptacle-opener, inventory-management, object-retriever, appliance-navigator, appliance-preparer, object-heater, location-navigator, object-placer, task-verifier

### 分支B — 红酒冷却

```
object-locator → receptacle-opener → object-picker
→ appliance-navigator(fridge) → object-cooler
→ location-navigator(sidetable) → object-placer → receptacle-closer
→ task-verifier
```

**用到的 skill（11个）：** object-locator, receptacle-opener, object-picker, appliance-navigator, appliance-preparer, object-cooler, location-navigator, receptacle-preparer, object-placer, receptacle-closer, task-verifier

### 分支C — 餐具清洗（盘→叉串行，与A/B/D并行）

```
object-locator → search-pattern-executor → object-picker(plate)
→ object-state-inspector → clean-object(sinkbasin)
→ location-navigator(diningtable) → object-placer

→ tool-locator(fork) → object-picker → clean-object → object-placer
→ task-verifier
```

> 盘子先找，叉子后找。两者都在分支C内串行，但分支C整体与A/B/D并行。

**用到的 skill（11个）：** object-locator, search-pattern-executor, object-picker, object-state-inspector, appliance-navigator, clean-object, location-navigator, receptacle-preparer, object-placer, tool-locator, task-verifier

### 分支D — 清理变质食物

```
environment-scanner(找腐烂物品) → object-locator
→ object-picker → receptacle-finder(garbagecan)
→ object-disposer → task-verifier
```

**用到的 skill（7个）：** environment-scanner, object-locator, object-picker, receptacle-finder, location-navigator, object-disposer, task-verifier

---

## 第三阶段：合并验证（串行）

```
task-verifier(综合检查 4 个子目标)
  ├─ ✅ diningtable: 2 热面包 + 1 净盘 + 1 净叉
  ├─ ✅ sidetable: 1 冰酒
  ├─ ✅ garbagecan: 已处理变质食物
  └─ ❌ 有缺失 → 触发 search-verifier 回溯重搜
```

---

## 并行度

| 阶段 | 并行数 | 说明 |
|---|---|---|
| 解析 | 1 | 必须串行 |
| 执行 | **4** | A/B/C/D 互不依赖 |
| C内部 | 1 | 叉子依赖先找到盘子 |
| 验证 | 1 | 汇总检查 |

---

## 资源冲突处理

| 冲突 | 处理方式 |
|---|---|
| 面包和酒都在冰箱 | A 先访问，B 复用已打开状态 |
| 面包/盘/叉都放餐桌 | 各自独立放置，顺序无关 |
| 某物品找不到 | object-locator 失败 → search-pattern-executor 穷举 → search-verifier 复检 |
| 冰箱最后谁关 | B（最后使用者）关闭 |
